# react-shopping-cart

## 설계 방향

1. 페이지 라우팅 x (싱글페이지)
2. 주문확인 페이지에서 새로고침시, 장바구니 페이지로 이동 (하나의 로직이기 때문에, 결제창 x)
3. 상태 -> 가공한다. (isChecked를 get으로 받아온 데이터와 조합)
